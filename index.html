프로젝트 개요

일정 관리 앱은 개인 계획만 보여주고, 버스 정보 앱은 교통 정보만 제공한다는 점에서 출발한 프로젝트입니다. 사용자가 자신의 일정을 등록하고 동시에 청주시 버스 번호를 검색해 노선, 정류장, 도착 정보를 확인할 수 있도록 Flask 기반 웹 서비스로 구성했습니다.

사용자와 관리자 기능

  • 일반 사용자는 회원가입, 로그인, 일정 등록·수정·삭제, 캘린더 화면을 통한 일정 확인을 사용할 수 있습니다.
  • 버스 메뉴에서는 버스 번호로 노선 ID를 조회하고, 경유 정류장 목록과 정류장별 실시간 도착 예정 정보를 확인할 수 있습니다.
  • 관리자는 사용자 목록, 사용자별 일정, 전체 일정, 로그인·로그아웃 같은 활동 로그를 확인할 수 있습니다.
  • Folium을 이용해 버스 노선 지도 HTML을 생성하는 기능도 포함되어 있습니다.

시스템 구성과 특징

핵심 구현은 `창업프로젝트/app.py`에 있으며 Flask 라우팅, 세션, 템플릿 렌더링, MySQL 데이터 관리, 공공데이터포털 버스 API 요청 로직이 함께 들어 있습니다. API 응답은 JSON을 우선 사용하고 실패 시 XML 파싱을 시도하도록 구성했습니다. 현재는 API 키 분리, 비밀번호 해시 처리, Java 클라이언트와 Flask 라우트 정합성 개선이 필요한 상태입니다.

웹캠 영상 분석과 Arduino RGB LED를 활용한 수질 상태 모니터링 시스템

프로젝트 개요

실제 수질 센서 없이 웹캠 영상만으로 물의 색상 변화를 감지하는 IoT 기반 교육용 프로토타입입니다. OpenCV로 카메라 프레임을 HSV 색공간으로 변환한 뒤 갈색, 초록색, 파란색 계열의 픽셀 비율을 비교해 오염 의심, 녹조 의심, 정상 상태를 판단합니다.

주요 구현 내용

  • 웹캠에서 실시간 프레임을 읽고 BGR 이미지를 HSV 색공간으로 변환합니다.
  • 갈색·초록색·파란색 범위에 대한 마스크를 생성하고 색상별 픽셀 합계를 계산합니다.
  • 임계값보다 픽셀이 적으면 감지 실패로 처리하고, 가장 많이 감지된 색상을 기준으로 상태를 분류합니다.
  • 분류 결과를 OpenCV 화면, 콘솔, `water_log.txt`, Arduino RGB LED로 동시에 확인할 수 있게 했습니다.

동작 기준과 한계

갈색 계열은 오염 의심으로 빨간색 점멸, 초록 계열은 녹조 의심으로 파란색 점멸, 파란 계열은 정상으로 초록색 점등을 수행합니다. 다만 색상 기반 판단이므로 조명, 카메라 성능, 배경색, 물통 재질에 따라 정확도가 달라질 수 있고, 실제 pH나 탁도 같은 화학적 수질 값은 직접 측정하지 않습니다.

로보테크 AI 과정에서 진행한 Python, 데이터 분석, TCP 통신, 로봇 관제, Arduino 실습 프로젝트 모음

프로젝트 개요

융합 로보테크 AI 자율주행로봇 개발자 과정 2기에서 진행한 실습과 팀 프로젝트를 모아둔 저장소입니다. 기초 Python 콘솔 게임부터 데이터 분석, TCP 멀티스레드 통신, TurtleBot3 관제 GUI, Arduino 안전 모듈, ROS2 기반 공기청정 로봇, 군집 협업 피킹·운반 로봇까지 과정 흐름에 따라 프로젝트를 나누어 정리했습니다.

포함된 프로젝트

  • 탑뷰 미로 탈출 게임: 텍스트 맵, 플레이어 이동, 몬스터 랜덤 이동, 아이템 적용, 스테이지 종료 조건을 구현했습니다.
  • 지역별 교통사고 분석: CSV 공공데이터를 pandas로 전처리하고 PyQt5 화면에서 조회·수정할 수 있게 구성했습니다.
  • TCP 소켓 기반 마피아 게임: 서버/클라이언트 구조, 다중 접속, 역할 배정, 낮/밤 페이즈, 투표와 처형 흐름을 구현했습니다.
  • 로봇 경로 탐색 관제 GUI: TurtleBot3 제어, Flask REST API, MySQL, PyQt5 관제 화면을 연결했습니다.
  • 자율주행자동차 보조안전모듈: 초음파, 온습도, 조도, 소음 센서 데이터를 UDP/HTTP로 PC GUI에 전달하고 LCD, LED, 부저로 위험 상황을 표시했습니다.

로봇 프로젝트 확장

TBPJ는 PMS7003과 ESP01에서 수집한 미세먼지 데이터를 MQTT로 전달하고, ROS2 노드가 오염도를 판단해 TurtleBot3 자율주행과 Spring Boot 웹 대시보드로 연결하는 프로젝트입니다. PACK은 구현 중인 군집 협업 피킹·운반 로봇으로, TurtleBot3 리더 로봇의 로봇팔 자동 피킹, ArUco 마커 인식, OpenCV solvePnP 위치 추정, MQTT·USB Serial 게이트웨이, F1/F2 추종 차량 FSM을 통합하는 방향으로 구성했습니다.

로보테크 AI 과정에서 진행한 ROS2 기반 LiDAR 모의 자율주행 및 데이터 저장 프로젝트

프로젝트 개요

실제 LiDAR 센서가 없는 환경에서도 자율주행 데이터 처리 흐름을 연습할 수 있도록 만든 ROS2 기반 모의 자율주행 프로젝트입니다. ROS2 Publisher가 0도부터 359도까지 360개의 거리 데이터를 생성해 `/scan` 토픽으로 발행하고, Python 구독 프로그램이 이 데이터를 받아 전방, 좌측, 우측의 평균 거리를 비교해 주행 방향을 판단합니다.

주요 구현 내용

  • `lidar_publisher.py`는 전방, 좌측, 우측 중 하나에 장애물이 있는 상황을 무작위로 만들고 2초 간격으로 LaserScan 메시지를 발행합니다.
  • `subscriber.py`는 rosbridge와 roslibpy를 통해 `/scan` 데이터를 구독하고, 안전 거리 기준에 따라 직진, 좌회전, 우회전을 결정합니다.
  • 판단 결과는 `geometry_msgs/Twist` 형태로 `/turtle1/cmd_vel`에 발행해 turtlesim 제어 흐름까지 확인할 수 있습니다.
  • `subscriber_db.py`는 LiDAR 원본 거리 데이터와 action 결과를 MySQL에 저장하고, `dataset_builder.py`는 JSON 형태의 거리 데이터를 angle_0부터 angle_359까지의 CSV 데이터셋으로 변환합니다.

학습 포인트

센서 메시지 생성, ROS 토픽 통신, Python 기반 구독, Rule 기반 주행 판단, MySQL 저장, CSV 전처리까지 이어지는 구조를 한 프로젝트 안에서 확인할 수 있습니다. 향후 실제 LiDAR 센서 연동, 장애물 회피 알고리즘 고도화, 학습 기반 모델 적용, 저장 데이터 시각화로 확장할 수 있습니다.

프로젝트 개요

C 언어로 구현한 콘솔 기반 영화 추천 및 관리 프로그램입니다. 사용자는 회원가입과 로그인을 통해 영화 목록을 확인하고, 영화 상세 정보 조회, 키워드 검색, 평점 등록을 할 수 있습니다. 관리자는 별도의 관리자 계정으로 로그인해 영화 정보를 추가, 수정, 삭제할 수 있습니다.

주요 기능

  • 사용자 기능으로 회원가입, 로그인, 전체 영화 목록 조회, 영화 상세 정보 확인, 제목 또는 장르 기반 키워드 검색, 영화 평점 등록을 제공합니다.
  • 관리자 기능으로 영화 추가, 영화 정보 수정, 영화 삭제, 영화 데이터 CSV 저장 기능을 제공합니다.
  • `Movie`, `User` 구조체를 사용해 영화와 사용자 정보를 메모리에 저장하고, `movie.csv`, `user.csv`, `review.csv`, `admin_log.csv` 같은 CSV 파일로 데이터를 관리합니다.
  • Visual Studio 프로젝트 파일과 콘솔 메뉴 구조를 포함해 C 언어 구조체, 파일 입출력, 문자열 처리, 메뉴 기반 UI를 함께 학습할 수 있습니다.

현재 구현 특징

성적 데이터를 학기별로 확인하고 검색·수정·삭제하는 Java Swing GUI 프로그램

프로젝트 개요

Java Swing으로 구현한 개인 성적 관리 GUI 프로그램입니다. 대학 성적표처럼 과목명, 년도/학기, 이수구분, 등급, 학점을 표 형태로 관리하며, 텍스트 파일에 저장된 성적 데이터를 불러와 학기별로 분류하고 JTable 기반 화면에서 확인할 수 있습니다.

주요 기능

  • `JFileChooser`로 성적 텍스트 파일을 선택하고, UTF-8로 읽은 뒤 공백 기준으로 과목 데이터를 파싱합니다.
  • `HashMap>` 구조를 사용해 년도/학기별로 성적을 묶어 관리합니다.
  • 추가, 수정, 삭제 버튼을 통해 프로그램 메모리 안의 성적 데이터를 관리하고, 검색어가 과목명, 학기, 이수구분, 등급, 학점 중 하나에 포함되면 검색 결과 팝업 테이블에 표시합니다.
  • 이전/다음 학기 버튼과 학기 선택 콤보박스를 함께 제공해 학기별 성적 탐색이 가능하도록 구성했습니다.

구현 특징과 개선 방향

핵심 구현은 `UniversityGradeManager.java`에 있으며 Swing `JFrame`, `JTable`, `JButton`, `JComboBox` 등을 활용합니다. 현재 추가, 수정, 삭제 결과를 원본 텍스트 파일에 다시 저장하는 기능은 구현되어 있지 않고, 과목명에 공백이 포함되면 파싱이 깨질 수 있습니다. 향후 표준 Java 프로젝트 구조 정리, Maven/Gradle 빌드 통합, 자동 저장, CSV 지원, 평균 평점 계산, 등급별 색상 표시 기능으로 확장할 수 있습니다.
